# PlayBall
Platform that allows all sports enthusiasts of all levels to discover more opportunities to interact with their associated sport

## Prerequisites
- **Rails:** `rails -v` >= 8.1.2

## Quickstart
1. Clone and install all dependencies
```
git clone git@github.com:gomezc08/JuegaLink.git
cd JuegaLink
uv pip install -r .\ml_service\requirements.txt
```

2. Run Rails application
```
cd backend
rails s
```
